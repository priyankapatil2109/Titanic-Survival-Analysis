# Titanic Survival Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to identify the factors affecting passenger survival.

## Dataset
- Titanic Dataset (train.csv)

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Features
- Data Cleaning
- Missing Value Handling
- Exploratory Data Analysis
- Data Visualization
- Correlation Analysis

## Visualizations
- Survival Count
- Survival by Gender
- Survival by Passenger Class
- Age Distribution
- Fare Distribution
- Correlation Heatmap
- Boxplot
- Pairplot

## Key Findings
- Female passengers had a higher survival rate than males.
- First-class passengers survived more frequently.
- Younger passengers had slightly better survival chances.
- Higher fares were associated with higher survival.
- Passenger class and gender were the strongest survival factors.
